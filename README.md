# clux

WezTermベースのAIエージェント協調型ターミナルマルチプレクサ

## Attribution

clux is based on [WezTerm](https://github.com/wezterm/wezterm) by Wez Furlong (MIT License).
Terminal emulation, font rendering, and multiplexer functionality are from WezTerm.
clux adds Claude Code coordination features via an embedded MCP server.
